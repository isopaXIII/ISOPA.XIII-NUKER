# 🚀 Overview

ISOPA.XIII NUKER is an automation tool designed to interact with server infrastructure and execute commands quickly.

The project focuses on providing a modular architecture allowing fast automation and command execution.

---

# ⚡ Features

* Modular automation system
* Fast command execution
* Server interaction utilities
* Easy deployment
* Lightweight architecture

# 📦 Requirements

Before installing the project make sure you have:

* Python **3.9+**
* Git

Install Git:

https://git-scm.com/

# ⬇ Installation

Clone the repository:

```bash id="8rqksf"
git clone https://github.com/isopaXIII/ISOPA.XIII-NUKER
cd ISOPA.XIII-NUKER
```

Install dependencies:

```bash id="g24xuk"
pip install -r requirements.txt
```

2. Reset your bot token
3. Update it in your configuration
